Finanças front-end

Projeto pessoal de desenvolvimento de um front-end para uma API de lançamentos financeiros.

Este projeto foi desenvolvido em conjunto com colegas do curso de ciência da computação, nos quais foram dividas as responsabilidades, de modo que o front-end foi desenvolvido por uma equipe e o backend por outra equipe. Ao final foi entregue uma aplicação que realiza lançamentos financeiros.
